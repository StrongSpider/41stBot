Privacy Policy

Usage of Data

The 41st Event Portal and its associated Discord bot use stored data, as defined below, for features including but not limited to: verifying users, tracking Roblox event participation, calculating event points, showing historical activity, and displaying Discord roles and usernames for authentication and permission purposes. No data is shared with any third-party service except Roblox (for username and group rank resolution) and Discord (for authentication and role validation). Data is never sold or used for advertising.

Stored Information

The application and bot may store the following information automatically when used:

  • Discord User IDs and usernames of server members
  • Discord member roles and nicknames
  • Roblox User IDs and usernames (for linked accounts)
  • Event participation data (event type, attendance, host/supervisor IDs, event points, and timestamps)
  • Session data required for Discord OAuth2 authentication (user ID, username, Discord avatar, officer status)
  • Historical logs of event edits for auditing

No other information outside of the above will be stored.

Updating Data

Stored data may be updated automatically when:

  • Members join or leave the server
  • Events are created, edited, or deleted
  • Users link or update their Roblox accounts
  • Permissions or roles change in Discord
The data may also be updated when using specific features of the application (e.g., manually editing an event).

Temporarily Stored Information

The application may keep the stored information in internal memory (cache) for a certain period to reduce requests to the database and external services (Discord/Roblox). Cached data is periodically discarded and re-acquired as needed. Data may also be dropped from cache if the bot is removed from the server.

Removal of Data

Automatic Removal

Stored data can be removed automatically by removing the bot from the server or removing members from the server via Discord moderation. If re-added, new data will be stored as described above.

Manual Removal

Manual data removal can be requested by emailing spiider.dev@gmail.com. Proof of server ownership will be required for manual removal requests. Only the Discord server owner may request manual removal, and requesting it will result in the bot being removed from the server (if still present).
